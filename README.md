# Back-Web-LingoLux

Este es un proyecto de API para aprender inglés.
Instalación

    Clona el repositorio
    git clone https://github.com/tu-usuario/api-ingles.

Instala las dependencias:

   npm install


Inicia el servidor de desarrollo:

    npm start
